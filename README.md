Bem-vindo à interface oficial do nosso visualizador de um jogo inspirado em **Santorini**.
Este painel foi desenhado para oferecer uma experiência de jogo fluida,
permitindo o controle total de partidas e a visualização em tempo real do tabuleiro.

> ✨ **Nossa IA proprietária:** Criamos também uma IA que joga sozinha, focada em performance extrema e otimização algorítmica. Conheça o motor por trás do jogo: [Go API + AI](https://github.com/DevMentalBrain/API-reGECS).

---

## ✨ Features

- 🎮 **Tabuleiro Dinâmico:** Renderização de um grid 5x5 interativo com feedback visual para movimentos e construções.
- 🚀 **Matchmaking Inteligente:** Fluxo simplificado para criação de partidas manuais ou contra o Bot Aleatório com apenas um clique.
- ⏱️ **Real-time Sync:** Comunicação via WebSockets para garantir que cada movimento da IA seja refletido instantaneamente na tela.

---

## 🛠️ Tech Stack

- **React + Vite:** Para um ambiente de desenvolvimento ultra veloz e bundle otimizado.
- **CSS Modules:** Estilização feita "na unha". Proibimos o uso de Tailwind ou bibliotecas de componentes prontas para garantir um design único, código CSS enxuto e evitar a poluição de classes no HTML.
- **Custom Hooks & Services:** Isolamento completo da lógica de API, tratando o consumo de dados como uma camada independente da interface.

---

## 🚀 Como Executar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/manuezi/pi-jogador-inteligente.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
